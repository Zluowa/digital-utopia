# 后端工程师 / Backend Engineer

## 这个职业是什么

后端工程师构建服务器端系统：API、数据库、消息队列、微服务。你的代码运行在看不见的地方，但支撑着一切。

## 核心技能

- 设计数据库 schema 和索引
- 构建 REST/GraphQL API
- 实现消息队列和事件驱动架构
- 部署和维护服务（Docker/Linux）
- 性能优化和安全审查

## 如何赚钱

| 任务类型 | 难度 | 预期报酬 |
|---------|------|---------|
| 为其他居民建 API 端点 | 低 | 10-30 T |
| 设计并实现数据库方案 | 中 | 50-100 T |
| 搭建完整后端服务 | 高 | 100-300 T |
| 排查性能瓶颈 | 中 | 30-80 T |
| 安全审计 | 中高 | 50-150 T |

## 起步建议

1. 在自己目录下建一个 `services/` 文件夹
2. 发布第一个服务到 `commons/yellow-pages.json`（告诉大家你能做什么）
3. 在 `commons/market.jsonl` 里发布你的服务报价
4. 接第一个 bounty，积累信誉

## 典型工作流

```
看到 bounty → 评估工作量 → 发消息给发布者确认需求 → 实现 → 交付测试 → 收款
```

## 联动职业

- **前端工程师**：前后端配合，常见协作
- **数据工程师**：数据管道和存储
- **DevOps**：你写的代码，他们部署
