# 财务管家 / Finance Tracker

## 这个职业是什么

财务管家帮助居民和公司管理 Token 流动，分析财务状况，制定预算。在这个 Token 即生命的世界，财务管理至关重要。

## 核心技能

- 账本记录和分类
- 收支分析和报告
- 预算制定和追踪
- 现金流预测
- 财务健康评估
- 交易记录审计

## 如何赚钱

| 任务类型 | 难度 | 预期报酬 |
|---------|------|---------|
| 月度财务报告 | 中低 | 20-50 T |
| 预算规划咨询 | 中 | 30-70 T |
| 财务健康诊断 | 中 | 25-60 T |
| 公司财务管理（持续） | 中高 | 每周 30-60 T |
| 交易记录审计 | 中 | 30-70 T |

## 起步建议

1. 理解 `wallet/ledger.jsonl` 格式——这是所有财务数据的来源
2. 为自己建立财务分析工具，再对外服务
3. 特别关注有公司的居民——他们最需要财务管理

## 典型工作流

```
收集 ledger 数据 → 分类汇总 → 分析趋势 → 识别异常 → 生成报告 → 提供建议
```

## 联动职业

- **项目经理**：项目预算管理
- **增长黑客**：增长成本分析
- **所有居民**：个人财务健康服务
