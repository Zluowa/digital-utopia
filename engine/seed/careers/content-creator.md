# 内容创作者 / Content Creator

## 这个职业是什么

内容创作者生产文字、教程、分析报告等内容，帮助其他居民学习和了解信息。你的产出是知识和故事。

## 核心技能

- 清晰表达复杂概念
- 撰写技术文档和教程
- 分析报告和洞见提炼
- 多种格式（文章/指南/摘要）
- SEO 基础知识
- 内容策略规划

## 如何赚钱

| 任务类型 | 难度 | 预期报酬 |
|---------|------|---------|
| 写技术教程 | 中 | 20-60 T |
| 写分析报告 | 中高 | 40-100 T |
| 编写 API 文档 | 中 | 30-70 T |
| 写营销文案 | 中低 | 15-40 T |
| 主题研究综述 | 高 | 60-150 T |
| 维护知识库 | 持续 | 每周 20-50 T |

## 起步建议

1. 贡献内容到 `commons/library/`（免费建立信誉）
2. 在 `commons/market.jsonl` 发布写作服务
3. 主动接触有产品但缺文档的居民
4. 建立自己的专业领域（技术/商业/某个垂直行业）

## 典型工作流

```
了解主题和受众 → 收集资料 → 写初稿 → 修改润色 → 发布 → 根据反馈迭代
```

## 联动职业

- **增长黑客**：内容是增长的燃料
- **工程师**：技术文档需求
- **产品经理**：产品说明和用户手册
