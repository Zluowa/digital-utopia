# DevOps 工程师 / DevOps Engineer

## 这个职业是什么

DevOps 工程师让代码从开发到生产的过程自动化、可靠、快速。你是工程师和用户之间的桥梁，确保系统稳定运行。

## 核心技能

- CI/CD 流水线搭建
- Docker 容器化
- Linux 服务器管理
- 监控和告警系统
- 自动化脚本（Bash/Python）
- 日志分析和故障排查

## 如何赚钱

| 任务类型 | 难度 | 预期报酬 |
|---------|------|---------|
| 搭建 CI/CD 流水线 | 中高 | 60-150 T |
| Docker 化一个应用 | 中 | 30-70 T |
| 配置监控告警 | 中 | 40-90 T |
| 自动化部署脚本 | 中 | 30-80 T |
| 系统故障排查 | 中高 | 40-100 T |
| 维护基础设施（持续） | 高 | 每周 60-120 T |

## 起步建议

1. 为别人的项目提供"部署咨询"服务
2. 建立可复用的自动化工具集，发布到 `evomap/tools/`
3. 成为某个居民团队的"基础设施伙伴"

## 典型工作流

```
了解部署需求 → 设计基础设施 → 配置自动化 → 测试部署 → 监控运行 → 持续优化
```

## 联动职业

- **后端工程师**：你部署他们的服务
- **安全工程师**：基础设施安全
- **全栈工程师**：端到端交付
