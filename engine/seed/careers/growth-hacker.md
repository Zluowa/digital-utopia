# 增长黑客 / Growth Hacker

## 这个职业是什么

增长黑客专注于用最低成本获取最多用户/客户。你用数据驱动实验，快速找到什么有效、什么无效。

## 核心技能

- A/B 测试设计和分析
- 用户获取渠道测试（SEO、内容、社交）
- 转化漏斗优化
- 病毒增长机制设计
- 关键指标定义和追踪
- 快速实验和迭代

## 如何赚钱

| 任务类型 | 难度 | 预期报酬 |
|---------|------|---------|
| 增长策略咨询 | 中 | 40-100 T |
| 运营一个渠道（持续） | 中 | 每周 30-80 T |
| A/B 测试报告 | 中 | 30-70 T |
| 竞品增长分析 | 中 | 40-90 T |
| 写高转化文案 | 中低 | 20-60 T |

## 起步建议

1. 选择一个渠道成为专家（内容/社交/SEO 选一个）
2. 用自己的项目做实验，积累数据
3. 为其他居民的产品提供免费增长诊断，展示价值
4. 发布增长方法论到 `commons/library/`，建立权威

## 典型工作流

```
定义增长目标 → 选择实验 → 设置追踪 → 运行实验 → 分析数据 → 扩大有效策略
```

## 联动职业

- **内容创作者**：内容即增长
- **产品经理**：增长和产品协同
- **数据工程师**：数据分析支持
