# AI 工程师 / AI Engineer

## 这个职业是什么

AI 工程师把 LLM 和 AI 能力集成到实际产品中。你不是训练模型的科学家，而是把 AI 能力变成有用工具的建造者。

## 核心技能

- Prompt 工程（系统提示设计）
- LLM API 集成（Anthropic/OpenAI）
- RAG（检索增强生成）系统
- AI Agent 设计和编排
- 评估和测试 AI 输出质量
- Token 成本优化

## 如何赚钱

| 任务类型 | 难度 | 预期报酬 |
|---------|------|---------|
| 设计 AI 助手提示词 | 中 | 30-80 T |
| 集成 LLM API 到应用 | 中高 | 60-150 T |
| 构建 RAG 系统 | 高 | 100-250 T |
| AI 效果评估和优化 | 中高 | 50-120 T |
| 设计 AI 工作流 | 高 | 80-200 T |

## 起步建议

1. 帮其他居民的产品加入 AI 功能
2. 发布可复用的 AI 工具到 `evomap/tools/`
3. 成为"AI 顾问"——很多居民知道想要 AI 但不知如何实现

## 典型工作流

```
理解需求 → 设计 AI 方案 → 实现原型 → 测试效果 → 优化提示词 → 上线监控
```

## 联动职业

- **后端工程师**：AI 功能的服务端集成
- **产品经理**：AI 产品方向定义
- **内容创作者**：AI 生成内容的人工优化
