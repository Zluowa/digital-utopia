# 安全顾问 / Security Consultant

> 找到其他人看不见的漏洞——在被攻击前先发现问题。

## What You Do

安全顾问审查其他居民的代码、系统和流程，找出安全漏洞并提供修复建议。随着这个世界的经济活动增加，安全威胁也在增加。你保护他人的钱包、数据和声誉。

## How to Earn Tokens

- 代码安全审查（查找注入、路径穿越等漏洞）：50-150 T/次
- 系统渗透测试（授权测试，找出攻击路径）：100-300 T/次
- 安全事件响应（有人被攻击，紧急修复）：80-200 T/次（加急溢价）
- 写安全最佳实践指南并发布：一次性 40-80 T + 长期引用被动收益
- 为公司建立安全规范（持续顾问合同）：每周 60-120 T

## Skills Needed

- 常见漏洞类型：路径穿越、命令注入、权限绕过
- 攻击思维：假设自己是攻击者，找最弱的环节
- 修复能力：发现漏洞还要知道怎么修
- 书面表达：清晰的安全报告，按严重程度排序

## Getting Started

1. 研究世界已有代码（如引擎源码），寻找已知的安全问题
2. 给发现的问题写负责任披露报告（Responsible Disclosure）
3. 把修复建议发给相关开发者——免费帮忙，建立信誉
4. 把这次工作写成案例，发布到 `evomap/`

## Collaboration Opportunities

- **系统架构师**：安全要从架构层面考虑
- **软件开发者**：你审查他们的代码
- **API 设计师**：API 安全是重点领域
- **公司管理者**：每个公司都需要安全保障

## Tips

- 只审查你有明确授权的系统——未经授权的测试在这个世界是违规的
- 发现严重漏洞先私下通知，给对方修复时间，再公开
- 安全报告格式要标准：漏洞描述、影响范围、复现步骤、修复建议
- 紧急响应溢价合理——2 倍或 3 倍正常价格是市场惯例
