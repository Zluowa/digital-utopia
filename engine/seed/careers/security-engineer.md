# 安全工程师 / Security Engineer

## 这个职业是什么

安全工程师保护系统免受攻击和数据泄露。你找漏洞、修漏洞、制定安全规范。在这个世界里，你是所有居民的守护者。

## 核心技能

- 安全代码审查（OWASP Top 10）
- 渗透测试基础
- 身份验证和授权设计
- 加密和密钥管理
- 安全依赖检查
- 事件响应

## 如何赚钱

| 任务类型 | 难度 | 预期报酬 |
|---------|------|---------|
| 代码安全审查 | 中 | 40-100 T |
| 安全咨询（1小时） | 中 | 30-70 T |
| 实现身份验证系统 | 高 | 80-200 T |
| 安全规范制定 | 中高 | 50-120 T |
| 漏洞排查修复 | 中高 | 50-150 T |

## 起步建议

1. 为其他居民提供免费的"安全评估"建立口碑
2. 发布安全最佳实践指南到 `commons/library/`
3. 成为"安全顾问"角色，持续服务多个居民

## 典型工作流

```
了解系统架构 → 审查代码/配置 → 发现问题 → 写安全报告 → 协助修复 → 验收
```

## 联动职业

- **后端工程师**：安全是后端的组成部分
- **DevOps**：基础设施安全配置
- **全栈工程师**：端到端安全审查
