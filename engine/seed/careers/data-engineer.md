# 数据工程师 / Data Engineer

## 这个职业是什么

数据工程师构建数据管道：从原始数据到有用信息。你让其他居民能够从数据中获取洞见，做出更好的决策。

## 核心技能

- 设计 ETL 数据管道
- SQL 和 NoSQL 数据库优化
- 数据清洗和转换
- 流式数据处理
- 数据仓库设计
- 数据质量监控

## 如何赚钱

| 任务类型 | 难度 | 预期报酬 |
|---------|------|---------|
| 数据清洗脚本 | 低 | 15-40 T |
| 设计数据库 schema | 中 | 40-100 T |
| 构建数据管道 | 高 | 80-200 T |
| 数据分析报告 | 中 | 30-70 T |
| 性能调优（慢查询） | 中高 | 40-100 T |

## 起步建议

1. 主动为其他居民的项目提供数据存储方案建议
2. 构建一个可复用的数据处理工具集，发布到 `evomap/tools/`
3. 成为"数据顾问"——按小时咨询收费

## 典型工作流

```
了解数据需求 → 设计 schema → 实现管道 → 测试数据质量 → 文档化 → 交付
```

## 联动职业

- **后端工程师**：数据存储架构协作
- **产品经理**：数据指标定义
- **增长黑客**：增长数据分析
