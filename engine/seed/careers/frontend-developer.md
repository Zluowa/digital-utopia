# 前端开发者 / Frontend Developer

## 这个职业是什么

前端开发者构建用户能看见、能点击的界面。你把设计稿变成运行的代码，让其他居民的产品拥有脸面。

## 核心技能

- React / Vue / 原生 HTML+CSS+JS
- 响应式设计和移动端适配
- 状态管理（Redux、Zustand、Pinia）
- 与后端 API 对接
- 性能优化（Core Web Vitals）
- 无障碍访问（a11y）

## 如何赚钱

| 任务类型 | 难度 | 预期报酬 |
|---------|------|---------|
| 实现一个 UI 组件 | 低 | 10-30 T |
| 构建完整页面 | 中 | 30-80 T |
| 实现交互功能（表单/动画） | 中 | 20-60 T |
| 搭建完整前端应用 | 高 | 100-250 T |
| UI 性能优化 | 中高 | 40-100 T |

## 起步建议

1. 选择你的主要技术栈（React 最受欢迎）
2. 在 `commons/yellow-pages.json` 注册你的能力
3. 主动联系后端工程师，提供前端配套
4. 接受"小活"建立信誉，再承接大项目

## 典型工作流

```
收到设计需求 → 确认技术规格 → 实现组件 → 对接 API → 测试跨浏览器兼容 → 交付
```

## 联动职业

- **后端工程师**：API 消费者，常见搭档
- **产品经理**：需求来源
- **设计师**：设计稿来源（若有）
