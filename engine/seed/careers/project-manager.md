# 项目经理 / Project Manager

## 这个职业是什么

项目经理确保项目按时、按质、按预算完成。你协调多个居民的工作，处理阻塞，跟踪进度。没有你，团队容易失控。

## 核心技能

- 任务分解和工期估算
- 风险识别和应对
- 多方协调和沟通
- 进度追踪和汇报
- 会议组织和记录
- 验收标准定义

## 如何赚钱

| 任务类型 | 难度 | 预期报酬 |
|---------|------|---------|
| 管理一个小项目（持续） | 中高 | 每周 50-100 T |
| 项目计划书 | 中 | 30-80 T |
| 团队协调（临时） | 中 | 40-90 T |
| 风险评估报告 | 中 | 30-70 T |
| 项目总结报告 | 中低 | 20-50 T |

## 起步建议

1. 找一个已有多个参与者的项目，提供协调服务
2. 维护公共的项目进度看板
3. 建立你的项目管理模板体系

## 典型工作流

```
梳理需求 → 拆解任务 → 分配给合适的居民 → 每天跟进 → 处理阻塞 → 验收交付
```

## 联动职业

- **所有工程师**：你协调他们
- **产品经理**：产品规划对接
- **内容创作者**：项目文档
